# Protect the Harvest
 
A wave-based survival game built in Unity. Defend your harvest from incoming enemies — aliens, zombies and more — using weapons, abilities and a fence system. After each stage you choose one of three upgrades to continue.
 
---
 
## Gameplay
 
- Stage-based enemy waves with variable types and stats
- Player controls a farmer with multiple weapons and abilities
- Enemies try to breach the fence and reach the harvest
- After clearing a stage, pick one of three upgrades
 
---

## ⚠️ About the Project Status
Development of the Protect the Harvest project has been suspended as of now. Although the project has not yet reached a fully completed version, the core mechanics and system architecture (Object Pooling, Manager structures, Weapon and Ability systems) have been largely implemented.

`Note: This repository is maintained to showcase the project’s current stage and the technical approaches used during the development process.`
